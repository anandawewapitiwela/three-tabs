This plugin adds a 'Three Tabs Widget' for your theme.

== Description ==

This plugin adds a 'Three Tabs Widget' for your theme. 
After activating the plugin go to the Widget section of your dashboard and drag  and drop the widget to your desired sidebar location.


== Installation ==

1. Download the plugin.
2. Install and Activate the plugin
3. Visit 'Widgets' in your theme appearance.
4. Drag and Drop the new widget to the desired sidebar.
5. Arrange the tabs as necessary.
6. Set the posts per page to your desired values.
7. Save your new settings.
8. Go to the front end of the site. You can see the new three tabs widget.
